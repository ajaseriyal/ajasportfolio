# Ajas — Portfolio

Personal portfolio site for **Ajas — Digital & IT Consultant**, featuring the SHIFTT Smart Biz Card product, services, and contact CTAs (WhatsApp / Email / Book / Call).

Adapted from a Blunar-style editorial layout: scroll-animated portrait, sticky case studies, and an interactive contact panel.

## Run locally

Open `index.html` directly, or serve this folder with any static server:

```powershell
python -m http.server 8080
```

Then visit `http://localhost:8080`.

No build step or package installation is required.

## Edit
- `index.html` — content & structure
- `styles.css` — visuals (palette, layout, SHIFTT card styling)
- `script.js` — scroll story, email reveal, micro-interactions
- `assets/` — images and fonts
