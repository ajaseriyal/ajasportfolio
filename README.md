# Blunar website clone

A responsive static recreation of the public portfolio at `blunar.cz`, including the immersive portrait scroll sequence, editorial client/about sections, sticky case studies, experiments grid, and interactive contact panel.

## Run locally

Open `index.html` directly, or serve this folder with any static server:

```powershell
python -m http.server 8080
```

Then visit `http://localhost:8080`.

No build step or package installation is required.
